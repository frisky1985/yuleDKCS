# yuleDKCS 数字钥匙完整功能矩阵

## 功能对比总览

| 功能模块 | Frontend (Web) | Mobile (SDK) | 说明 |
|---------|---------------|--------------|------|
| **钥匙列表** | ✅ 管理视图 | ✅ 本地列表 | Web管理所有钥匙，App显示本地钥匙 |
| **钥匙详情** | ✅ 完整信息 | ✅ 简要信息 | Web查看详细信息，App快速识别 |
| **钥匙分享** | ✅ 发起分享 | ✅ 接收/查看 | Web发起分享，App接收并使用 |
| **权限管理** | ✅ 设置/修改 | ✅ 显示权限 | Web配置权限策略，App显示当前权限 |
| **使用记录** | ✅ 历史查询 | ✅ 本次记录 | Web查看所有记录，App保存本次使用 |
| **激活/停用** | ✅ 控制开关 | ✅ 状态显示 | Web控制钥匙状态，App根据状态启用 |
| **撤销钥匙** | ✅ 撤销操作 | ✅ 删除本地 | Web从服务器撤销，App删除本地数据 |
| **二维码展示** | ✅ 生成/展示 | ✅ 扫码解析 | Web生成分享码，App扫描激活 |
| **BLE连接** | ❌ 不支持 | ✅ 核心功能 | 只有App能BLE连接车辆 |
| **车辆控制** | ❌ 不支持 | ✅ 控制命令 | 解锁/锁车/启动等命令通过BLE发送 |
| **离线使用** | ❌ 需要网络 | ✅ 本地存储 | App支持无网络时BLE控车 |
| **实时状态** | ✅ 在线状态 | ✅ BLE状态 | Web看云端状态，App看BLE连接状态 |
| **钥匙认证** | ✅ 初始激活 | ✅ SE验证 | Web完成认证流程，App使用SE安全元 |
| **批量操作** | ✅ 批量撤销 | ❌ 单条操作 | Web支持批量管理 |

## 详细功能设计

### 1. Frontend (Web管理后台)

#### 1.1 钥匙列表管理
- 分页展示所有钥匙
- 按状态筛选(有效/过期/已撤销)
- 按协议类型筛选(CCC/ICCOA/ICCE)
- 关键词搜索
- 批量选择操作

#### 1.2 钥匙分享功能
- 通过邮箱/手机号分享
- 设置有效期(1天/7天/30天/自定义)
- 精细化权限控制(解锁/锁车/启动/后备箱/窗户)
- 使用次数限制
- 时间段限制(例如只能白天使用)
- 地理位置限制(包围栏功能)

#### 1.3 使用记录查询
- 按时间范围查询
- 按操作类型筛选
- 地图展示操作位置
- 操作详情弹窗
- 导出CSV报表

#### 1.4 钥匙激活/停用
- 一键停用(不删除，仅禁用)
- 重新激活
- 设置自动过期
- 到期提醒

#### 1.5 二维码功能
- 生成钥匙分享码
- 扫码激活钥匙
- 临时授权码(一次性)

---

### 2. Mobile (SDK)

#### 2.1 钥匙生命周期
- 接收分享的钥匙
- 激活钥匙(与SE绑定)
- 查看钥匙列表
- 显示钥匙详情
- 删除本地钥匙

#### 2.2 BLE通信
- 扫描附近车辆
- 自动连接(靠近触发)
- 手动连接
- 连接状态监听
- 断线重连

#### 2.3 车辆控制命令
- 解锁(`Command.Unlock`)
- 锁车(`Command.Lock`)
- 启动引擎(`Command.StartEngine`)
- 停止引擎(`Command.StopEngine`)
- 开后备箱(`Command.OpenTrunk`)
- 开窗(`Command.OpenWindows`)
- 关窗(`Command.CloseWindows`)
- 寻车(`Command.FindVehicle`)

#### 2.4 安全认证
- SE安全元验证
- 生物识别确认(指纹/面容)
- PIN码确认
- 二次确认

#### 2.5 离线支持
- 钥匙数据本地加密存储
- 无网络时BLE通信
- 操作记录本地缓存
- 网络恢复后同步

#### 2.6 实时状态
- 车辆连接状态
- 门锁状态
- 引擎状态
- 电量/油量
- 警报状态

## 技术实现要点

### Frontend 技术栈
- React 18 + TypeScript
- Material-UI (MUI) v5
- React Query (数据获取)
- Zustand (状态管理)
- Recharts (数据可视化)
- React Router v6 (路由)

### Mobile 技术栈

#### Android
- Kotlin Coroutines + Flow
- BluetoothLeScanner (BLE扫描)
- BluetoothGatt (BLE通信)
- Android Keystore (SE安全元)
- BiometricPrompt (生物识别)
- Retrofit (HTTP通信)

#### iOS
- Swift Concurrency (async/await)
- CoreBluetooth (BLE)
- LocalAuthentication (FaceID/TouchID)
- CryptoKit (加密)
- Keychain Services (安全存储)
- URLSession (HTTP)

## 使用场景示例

### 场景1: 车主分享钥匙给朋友
```
1. 车主打开 Web 后台 (frontend)
2. 选择要分享的车辆钥匙
3. 输入朋友手机号
4. 设置权限:只允许解锁/锁车，禁止启动
5. 设置有效期: 3天
6. 点击分享
7. 朋友收到短信/邮件通知
8. 朋友下载 App，注册登录
9. App 中接收并激活钥匙 (mobile)
10. 朋友接近车辆，App 自动解锁
```

### 场景2: 离线使用钥匙
```
1. 用户接近地下室停车场（无手机信号）
2. App 自动扫描附近车辆
3. 通过 BLE 建立连接
4. 进行 SE 验证
5. 验证通过后发送解锁命令
6. 操作记录保存在本地
7. 离开车辆后自动锁车
8. 有网络时同步操作记录到云端
```
