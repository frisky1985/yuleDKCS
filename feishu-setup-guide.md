# 飞书机器人配置向导
# 请按照以下步骤操作

## 步骤 1: 创建飞书应用

1. 访问 https://open.feishu.cn/app
2. 点击「创建企业自建应用」
3. 填写应用名称："OpenClaw 智能助手"（或其他你喜欢的名字）
4. 选择应用类型："企业内部应用"
5. 点击「创建应用」

## 步骤 2: 添加机器人能力

1. 在应用详情页左侧菜单，点击「添加能力」
2. 找到「机器人」并点击「添加」
3. 机器人已添加到应用

## 步骤 3: 配置权限

1. 点击左侧「权限管理」
2. 搜索并添加以下权限：
   - ✅ `im:message:send_as_bot` - 以机器人身份发送消息
   - ✅ `im:message` - 发送消息
   - ✅ `im:message.group_at_msg` - 接收群聊@消息
   - ✅ `im:message.p2p_msg` - 接收私聊消息
   - ✅ `im:chat:readonly` - 读取群聊信息
   - ✅ `im:chat.members:read` - 读取群成员信息

## 步骤 4: 配置事件订阅

1. 点击左侧「事件订阅」
2. 在「长连接」区域，点击「添加事件」
3. 选择：`im.message.receive_v1`（接收消息事件）
4. 保存配置

## 步骤 5: 获取凭证

1. 点击左侧「凭证与基础信息」
2. 复制以下信息：
   - **App ID** (格式: cli_xxxxxxxxxxxxxxxx)
   - **App Secret** (点击显示并复制)

## 步骤 6: 发布应用

1. 点击左侧「版本管理与发布」
2. 点击「创建版本」
3. 填写版本号（如 1.0.0）和更新说明
4. 点击「保存并发布」
5. 申请发布（需要企业管理员审批）

## 步骤 7: 将机器人添加到群聊

1. 在飞书客户端，进入目标群聊
2. 点击群设置 → 群机器人 → 添加机器人
3. 搜索你创建的应用名称
4. 点击添加

## 步骤 8: 获取群聊 ID

添加机器人后，在群里发送一条消息，机器人会收到事件。
或者使用以下方法获取：

方法 A: 通过飞书开放平台 API 查询
方法 B: 在 bridge 启动后查看日志中的 chat_id
