compliance_rules (合规规则表)

核心规则定义，支持多协议版本

字段名 类型 约束 说明
rule_id UUID PK 规则唯一标识
rule_code VARCHAR(50) UK, NOT NULL 规则编号 (如: CCC-DK3-SEC-001)
protocol VARCHAR(20) CHECK IN ('CCC','ICCOA','ICCE') 归属协议
severity VARCHAR(20) CHECK IN ('mandatory','recommended','optional') 强制性级别
validation_logic JSONB NOT NULL 验证逻辑定义 (可执行配置)

test_tasks (测试任务表)

测试执行实例，支持流水线状态追踪

字段名 类型 约束 说明
task_id UUID PK 任务唯一标识
target_id UUID FK > test_targets 测试对象
status VARCHAR(30) DEFAULT 'pending' pending/running/completed/failed
progress INTEGER 计算字段 completed/total

test_results (测试结果表)

按月分区，存储详细验证结果和证据

🔧 分区策略: PARTITION BY RANGE (created_at) - 按月分区，支持历史数据归档
字段名 类型 约束 说明
result_id UUID PK 结果唯一标识
result_status VARCHAR(20) CHECK IN 7状态 pass/fail/warning/skip/error/inconclusive
evidence_files JSONB DEFAULT '[]' 证据文件路径列表
log_data TEXT - 详细测试日志

packet_captures (抓包记录表)

实时通信抓包，支持BLE/NFC/UWB

字段名 类型 约束 说明
capture_id UUID PK 抓包唯一标识
protocol_layer VARCHAR(30) - ble/nfc/uwb/mqtt/https
file_path VARCHAR(500) - pcapng/pcap 文件路径
packet_count INTEGER - 包数量统计
